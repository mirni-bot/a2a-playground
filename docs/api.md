# API

> PLAYGROUND_MARKER

This is sample documentation for the A2A playground. It exists so
`get_file_contents` and `search_code` have some non-code content to return.

## Endpoints (hypothetical)

| Method | Path | Description |
|--------|------|-------------|
| GET | `/hello` | Return a greeting |
| POST | `/hello` | Accept a name and return a greeting |

See [examples/](../examples/) for reference implementations in Python, JS, Go, and Rust.
